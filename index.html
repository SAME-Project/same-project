    SAME Project

    Easily Turn Jupyter Notebooks Into Pipelines



    Install: pip3 install sameproject
    Run in a folder with a notebook:
    same init
    sets up requirements.txt and same.yaml
    same verify
    checks dependencies (optional, requires Docker)
    same run
    deploy a pipeline! (requires Kubeflow Pipelines)
